# Practices

Each folder is a self-contained interview practice. Open one and read its own
`README.md` for the task. See the [root README](../README.md) for conventions and how
to run a practice.

| Practice | What it is | Kind |
|---|---|---|
| [`schedulr-python`](./schedulr-python) | Calendly-style booking service | Debug & harden |
| [`fileshare-python`](./fileshare-python) | WeTransfer-style file sharing | Debug & harden |
| [`linklock-python`](./linklock-python) | Magic-link / passwordless auth | Debug & harden |
| [`transcribe-python`](./transcribe-python) | Whisper-style transcription job queue | Debug & harden |
| [`gridbot-any`](./gridbot-any) | Robot-on-a-grid simulator | Build from scratch |
| [`lru-cache-any`](./lru-cache-any) | O(1) LRU cache | Build from scratch |

A `-python` suffix means the practice is set up for Python; `-any` means any language.
